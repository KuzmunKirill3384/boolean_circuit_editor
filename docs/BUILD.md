# Сборка и запуск

Документ фиксирует минимальные правила подготовки окружения для команды. Пока в репозитории нет кода приложения, этот файл нужен как договоренность о том, как проект будет запускаться после начала реализации.

## Базовые требования

- `Python 3.10+`
- `venv`
- `pip`
- GUI: `PyQt6` как основной вариант

## Подготовка окружения

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Для Windows:

```bash
.venv\Scripts\activate
pip install -r requirements.txt
```

## Планируемые команды

После появления кода проект должен поддерживать как минимум:

- запуск приложения;
- запуск тестов;
- при необходимости запуск форматирования или проверки стиля.

Базовый целевой сценарий:

```bash
python run.py
pytest
```

## Что должно быть описано к моменту сдачи

- точная команда запуска приложения;
- точная команда запуска тестов;
- версии основных библиотек;
- особенности запуска на Windows и Linux/macOS;
- известные ограничения.
