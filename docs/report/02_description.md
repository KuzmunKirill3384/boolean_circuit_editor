# Описание приложения

Приложение «Редактор булевых схем» представляет собой графическую среду для создания и анализа цифровых логических схем. Пользователь может добавлять на рабочую область элементы:

- **INPUT** – вход схемы (0 или 1)
- **OUTPUT** – выход схемы
- **AND** – логическое И
- **OR** – логическое ИЛИ
- **XOR** – исключающее ИЛИ
- **EQUAL** – логическое равенство

Элементы соединяются между собой проводами. Приложение проверяет корректность соединений (нельзя создать цикл, нельзя подключить больше входов, чем у элемента). Все операции редактирования поддерживают отмену (Undo) и повтор (Redo).

## Основные возможности

1. **Редактирование схемы**  
   – Добавление, удаление, перемещение элементов.  
   – Создание и удаление соединений.  
   – Undo/Redo для всех действий.

2. **Таблица истинности**  
   – Для всей схемы: отображает все комбинации входных сигналов и соответствующие значения на выходах.  
   – Для выбранного элемента: показывает, как зависит значение выбранного узла от входов схемы; при этом влияющие на него узлы подсвечиваются на схеме.

3. **Упрощение схемы**  
   – Пользователь может задать фиксированные значения для одного или нескольких входов.  
   – Программа оценивает, сколько логических элементов может быть удалено при таком упрощении.  
   – По команде «Упростить» схема заменяется на эквивалентную, но с меньшим числом элементов.

4. **Полиномиальное представление**  
   – Для любого элемента (или для всей схемы) выводится полином Жегалкина – представление булевой функции в виде суммы по модулю 2 произведений переменных.

5. **Настройки отображения**  
   – Возможность изменять цвета элементов, линий, фона, размеры узлов и шрифт подписей. Настройки сохраняются между сеансами работы.

6. **Сохранение и загрузка**  
   – Схема сохраняется в XML-файл, включая координаты элементов и связи.  
   – Загрузка ранее сохранённой схемы позволяет продолжить редактирование.
