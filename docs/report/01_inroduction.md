# Введение

Целью данной работы является разработка приложения для создания, редактирования и анализа булевых схем. Приложение позволяет пользователю конструировать логические схемы из базовых элементов (AND, OR, XOR, EQUAL), проверять их корректность, вычислять таблицу истинности, упрощать схему при фиксированных входах, получать полиномиальное представление элементов, а также сохранять и загружать схемы в формате XML.

Работа выполнена в рамках курса [название курса] с использованием современных подходов к разработке программного обеспечения: применение архитектурных паттернов, тестирование, документирование.

## Задачи

1. Спроектировать модель булевой схемы с использованием паттерна Composite.
2. Реализовать возможность редактирования с поддержкой Undo/Redo через паттерны Command и Visitor.
3. Обеспечить валидацию схемы (проверка циклов, корректности соединений).
4. Реализовать вычисление таблицы истинности для всей схемы и для выбранного элемента.
5. Реализовать упрощение схемы при фиксированных значениях на входах.
6. Реализовать вычисление полиномиального представления (полином Жегалкина) для каждого элемента.
7. Разработать графический интерфейс для удобной работы со схемой.
8. Реализовать сохранение и загрузку схем в XML.
9. Написать тесты для всех модулей и подготовить документацию.
