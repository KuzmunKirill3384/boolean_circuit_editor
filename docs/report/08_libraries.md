# Используемые библиотеки

Приложение разработано на языке Python 3.10+ и использует минимальный набор сторонних библиотек для обеспечения стабильности, кроссплатформенности и простоты развёртывания.

| Библиотека | Версия | Назначение |
|------------|--------|------------|
| Python | 3.10+ | Язык программирования и стандартная библиотека |
| PyQt6 | 6.4+ | Графический интерфейс (виджеты, графика, события) |
| pytest | 7.0+ | Фреймворк для модульного и интеграционного тестирования |
| xml.etree.ElementTree | (входит в Python) | Парсинг и генерация XML-файлов схем |

## Обоснование выбора

### Python 3.10+
- Кроссплатформенность: код работает на Windows, Linux, macOS без изменений.
- Большая стандартная библиотека, покрывающая большинство потребностей (работа с XML, файловая система, типы данных).
- Простота изучения и поддержки, что важно для учебного проекта.

### PyQt6
- **Мощный графический фреймворк**: `QGraphicsScene` / `QGraphicsView` идеально подходят для реализации рабочей области с элементами и связями.
- **Богатый набор виджетов**: таблицы, выпадающие списки, диалоги, док-панели – всё необходимое для интерфейса.
- **Кроссплатформенность**: Qt работает на всех целевых ОС.
- **Лицензия**: GPL/LGPL, что допустимо для учебного проекта.
- **Альтернатива Tkinter**: PyQt6 выбран вместо встроенного Tkinter из-за лучшей поддержки графических сцен, современного внешнего вида и удобства работы со сложными UI.

### pytest
- **Простота**: минимальный код для написания тестов, мощные фикстуры.
- **Расширяемость**: поддержка плагинов (например, `pytest-cov` для покрытия кода).
- **Удобство запуска**: одна команда для всех тестов.
- Широко используется в сообществе Python.

### xml.etree.ElementTree
- **Входит в стандартную библиотеку**: не требует установки дополнительных пакетов.
- **Достаточен для задач**: наше приложение не требует сложной валидации XSD или XPath, поэтому `ElementTree` полностью покрывает потребности в чтении/записи XML.
- **Альтернатива lxml**: более лёгкий, без внешних зависимостей.

## Дополнительные библиотеки (не используются)

В проекте сознательно отказались от использования дополнительных библиотек, таких как:
- **lxml** – для упрощения установки и уменьшения размера зависимостей.
- **numpy/sympy** – для вычислений; таблица истинности и полиномы реализованы на чистом Python, что достаточно для учебных схем.
- **matplotlib** – не требуется для визуализации, так как все графики (таблицы) выводятся встроенными средствами Qt.

## Установка

Все зависимости перечислены в файле `requirements.txt`:

PyQt6>=6.4
pytest>=7.0

Установка выполняется командой:
```bash
pip install -r requirements.txt
```

Стандартная библиотека Python (включая `xml.etree.ElementTree`) не требует дополнительной установки.
