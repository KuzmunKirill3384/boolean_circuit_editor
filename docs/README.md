# Документация проекта

В этой папке лежат все организационные и проектные документы.

## Основные документы

- `WORK_PLAN.md` — распределение задач между участниками
- `PRODUCTION_PLAN.md` — подробный производственный план
- `ARCHITECTURE.md` — внутренняя архитектура репозитория
- `FILE_FORMAT.md` — спецификация XML
- `BUILD.md` — правила сборки и запуска
- `TEST_PLAN.md` — стратегия тестирования

## Подпапки

- `report/` — заготовки и материалы для итогового отчета
- `uml/` — диаграммы классов и последовательности
- `screenshots/` — скриншоты для руководства оператора и отчета
