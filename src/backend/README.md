Зона ответственности: **Кузьмин Кирилл Владимирович**.

В этом каталоге будет жить вся прикладная и доменная логика, не зависящая от GUI.

## Подмодули

- `model/` — структура схемы и элементы
- `validation/` — проверки корректности
- `commands/` — команды и история undo/redo
- `visitor/` — обход модели
- `io/` — XML Builder
- `logic/` — таблица истинности, упрощение, полиномы
- `application/` — фасад для фронтенда
