# Исходный код

Здесь будет лежать исходный код приложения после старта реализации.

## Разделение

- `backend/` — модель схемы, команды, XML и вычислительная логика
- `frontend/` — GUI и пользовательское взаимодействие
- `common/` — общие контракты и соглашения

Код пока не добавлялся. Внутри каждого каталога есть README с назначением модуля.
