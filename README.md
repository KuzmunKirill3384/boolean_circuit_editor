# Редактор булевых схем

Приложение для создания, редактирования и анализа булевых схем из логических элементов `AND`, `OR`, `XOR`, `EQUAL` с поддержкой таблицы истинности, упрощения, полиномиального представления и сохранения схем в XML.

## Текущий статус

Репозиторий подготовлен как **архитектурный каркас**:

- развернута внутренняя структура каталогов;
- добавлены документы по архитектуре, планированию, формату XML, сборке и тестированию;
- код пока **не добавлялся**, чтобы команда могла спокойно стартовать по согласованной структуре.

## Основные возможности проекта

- **Редактирование схемы**: добавление элементов, создание связей, проверка корректности структуры, Undo/Redo.
- **Таблица истинности**: для всей схемы и для отдельного узла с подсветкой влияющих элементов.
- **Упрощение схемы**: при известных значениях входов с оценкой числа удаляемых элементов.
- **Полиномиальное представление**: вывод полинома для логических элементов.
- **Настройки отображения**: цвета, размеры, шрифты и другие параметры визуализации.
- **Сохранение и загрузка**: обмен схемами через XML.

## Стек

- `Python 3.10+`
- `PyQt6` как основной вариант GUI
- `Tkinter` как допустимая альтернатива
- `pytest` для тестирования
- `xml.etree.ElementTree` или `lxml` для XML

## Архитектурные паттерны

- **Composite**: модель схемы и иерархия элементов
- **Command**: операции редактирования с поддержкой отмены
- **Visitor**: обход модели при do/undo и вычислениях
- **Factory**: создание команд
- **Builder**: загрузка и сохранение XML
